8. 动作模板库
访问方法	get
访问地址	restapi/plan/coach/plandesc
访问参数	token

返回		所有动作
{
	status: 200
	desc: 动作数组
	[{
		id: 动作id long
		duration: 休息时间 int
		name: 训练名字
		intro: 简介
		category: 上传媒体的类型 0图片 1视频。该字段的意义在于，视频和师范图片只能上传一种，且只需要传一张图
		pic: 缩略图， 需单独上传，放师范图片旁边，
		url: 当类型是图片时为示范图的url，视频时为视频url，视频时不需要预览。
		quantiry: int 数量
		units: 单位 次数/时长
	}]
}

8. 添加动作模板
访问方法	post
访问地址	restapi/plan/coach/plandesc/add
访问参数	token
			name	
			intro
			pic File类型
			picType 图片类型 jpg等
			media File 类型
			mediaType 类型
			category  视频1 或 图片0 int
			quantiry 数量
			units 单位 次数/时长 传汉字即可
			duration 时长

返回		
{
	status: 200
	失败返回 400 
			 message:"没有权限操作"
}
